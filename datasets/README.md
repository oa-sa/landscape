## Notes

- License versions vary. Always check the individual dataset metadata for the exact license.
- "—" in the Format column means the format was not confirmed during our audit.
- Datasets marked as statistical/reporting contain aggregate numbers, not individual service locations or contact details.
- All information is as of April 2026.

# Licenses in Australian Open Data

A guide to the licenses used across Australian government open data portals.

## Common Licenses

### Creative Commons Attribution 4.0 International (CC-BY 4.0)

- **Used by**: VIC, QLD, some NSW, some Federal datasets
- **You can**: Copy, redistribute, adapt, use commercially
- **You must**: Give appropriate credit, link to the license, indicate changes
- **You cannot**: Add restrictions that prevent others from using the data
- **Full text**: https://creativecommons.org/licenses/by/4.0/

**Attribution format (recommended)**:
```
Contains [Dataset Name] © [Agency Name], [Year].
Licensed under CC BY 4.0: https://creativecommons.org/licenses/by/4.0/
```

### Creative Commons Attribution 3.0 Australia (CC-BY 3.0 AU)

- **Used by**: Federal (data.gov.au), AIHW, AURIN datasets
- **Functionally identical** to CC-BY 4.0 but specific to Australian jurisdiction
- Same permissions and requirements as CC-BY 4.0
- **Full text**: https://creativecommons.org/licenses/by/3.0/au/

### Creative Commons Attribution 2.5 Australia (CC-BY 2.5 AU)

- **Used by**: Some Dept of Social Services datasets
- Older version, same core permissions as above
- **Full text**: https://creativecommons.org/licenses/by/2.5/au/

### CC0 1.0 Universal (Public Domain)

- **Used by**: Rare in Australian government data
- **You can**: Do anything — no attribution required
- **Full text**: https://creativecommons.org/publicdomain/zero/1.0/

---

## Less Common Licenses

### Other (Open)

- **Used by**: City of Casey, some council datasets
- Indicates the data is open but uses custom terms rather than a standard Creative Commons license
- **Action required**: Check the specific dataset page for the exact terms

### Custom (Active Acceptance)

- **Used by**: Some WA datasets
- May require clicking through or agreeing to specific terms before use
- **Action required**: Review terms on the dataset page before using

### Not Specified

- **Used by**: Various datasets across portals
- No license information provided
- **Action required**: Contact the publishing agency to clarify usage rights before redistributing

## What This Means in Practice

| License | Can Use | Can Adapt | Can Redistribute | Must Attribute | Can Commercialise |
|---------|---------|-----------|-----------------|----------------|-------------------|
| CC-BY 4.0 | Yes | Yes | Yes | Yes | Yes |
| CC-BY 3.0 AU | Yes | Yes | Yes | Yes | Yes |
| CC-BY 2.5 AU | Yes | Yes | Yes | Yes | Yes |
| CC0 1.0 | Yes | Yes | Yes | No | Yes |
| Other (Open) | Likely | Check terms | Check terms | Check terms | Check terms |
| Custom | Check terms | Check terms | Check terms | Check terms | Check terms |
| Not Specified | Unknown | Unknown | Unknown | Unknown | Unknown |

## OpenStreetMap Data

OpenStreetMap data uses the **Open Database License (ODbL)**:

- **You can**: Use, adapt, and redistribute
- **You must**: Attribute ("© OpenStreetMap contributors") and share derivative databases under ODbL
- **Full text**: https://opendatacommons.org/licenses/odbl/

---

## Notes

- Always verify the license on the individual dataset page before use. Licenses listed here reflect what was documented at the time of audit (April 2026).
- When combining data from multiple sources with different CC-BY versions, attribute each source individually.
- CC-BY licenses are compatible with each other — data under CC-BY 3.0 AU can be combined with data under CC-BY 4.0.
